# docker-tcconfig
Docker image for [tcconfig]( https://github.com/thombashi/tcconfig )

```
docker pull thombashi/tcconfig

docker run -d --cap-add NET_ADMIN -t thombashi/tcconfig
```


# Built-in Packages
- iperf
- iputils-ping
- [tcconfig]( https://github.com/thombashi/tcconfig )
